## 3 Simple Steps to Get Going

### Step 1: Test It Locally (Optional)
```bash
npm run electron-dev
```
This launches the app on your computer for testing.

### Step 2: Build the `.exe` File
```bash
npm run electron-build-portable
```
Creates: `dist/Team Schedule Generator-1.0.0-portable.exe`

### Step 3: Share with Your Team
Send them the `.exe` file from the `dist/` folder. They can:
- Download it
- Double-click it
- Use it!
